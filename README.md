## 介绍

个人项目脚手架，方便快速创建项目。

## 快速开始

```bash
# npm
npm create vfan@latest

# pnpm
pnpm create vfan@latest

# yarn
yarn create vfan@latest
```

## 模板总览

- vue2-js
- vue3-ts
- react18-ts

**💡 提示：** 不常用的模板统一放置在 [`uncommon-templates`](https://github.com/VfanLee/create-vfan/blob/main/uncommon-templates) 目录下。
