## 介绍

将个人常用的项目模板做成脚手架，方便快速创建项目。

## 模板总览

- vue2-js
- vue3-ts
- react18-ts

> [!TIP]
> 不常用的模板统一放置在 [`uncommon-templates`](https://github.com/VfanLee/create-vfan/blob/main/uncommon-templates) 目录下。
